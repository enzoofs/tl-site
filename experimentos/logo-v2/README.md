# Logo TimeLabs — V2 (baseado nos rascunhos)

Traduzindo os conceitos dos rascunhos em SVG. Abre no navegador:
```bash
firefox ~/projetos/pessoal/design-system/experimentos/logo-v2/*.svg
```

## 01 — Hexagonos Empilhados (TIME/LABS inscritos)
`01-hex-empilhado.svg`
Dois hexagonos compartilhando aresta. TIME no superior, LABS no inferior.
Tres amarelos contam a historia:
- **Entrada** (aresta superior-esquerda dourada): input do processo
- **Nucleo** (aresta compartilhada, amarelo forte): a reacao alquimica
- **Saida** (aresta inferior-direita dourada): output otimizado

Fluxo diagonal descendente = transformacao.

## 02 — Monograma TL via Hexagonos
`02-monograma-tl-hex.svg`
Dois hex que *viram* as letras (conforme rascunho "cabeca do T / corpo do L"):
- **T**: topo do hex eh a barra, haste amarela no eixo vertical (nucleo)
- **L**: aresta esquerda + base formam o L, nucleo amarelo vertical

Ao lado: wordmark "time labs" em pesos diferentes pra criar hierarquia.

## 03 — Silhueta Minimalista
`03-silhueta-nucleo.svg`
Simplificacao extrema (conforme 3o rascunho): so arco superior + arco inferior
sem as arestas laterais, criando uma forma etera. Nucleo amarelo vertical
conecta os dois arcos = reator alquimico visto de lado.

Mais abstrato, funciona como selo.

## 05 — LOGOMARCA T+S via arestas (baseado rascunho)
`05-logomarca-arestas.svg`
**Esta eh a direcao.** Letras construidas inteiramente a partir de arestas
de hexagonos empilhados. Outline dos hex visivel sutil pra mostrar a grade
que gerou a forma.

- **T**: barra superior = aresta de topo + dois bracos diagonais descendo
  (top-right + top-left). Haste vertical amarela percorre o eixo central
  dos dois hexes (nucleo).
- **S**: zigzag de 7 arestas consecutivas entre os dois hexes empilhados —
  topo esq → diagonal desc-esq → retorno diagonal desc-dir → aresta central
  dir → diagonal desc-dir → retorno diagonal desc-esq → base esq.
  Aresta compartilhada (central horizontal) em amarelo = nucleo do S.

Ambas letras seguem a MESMA grammar: arestas sao a tipografia. Amarelo
pontua o "coracao" de cada letra. Resultado: tipografia hexagonal nativa
da marca, impossivel de confundir com outras.

## 04 — LOGOMARCA T+S (feedback Enzo)
`04-logomarca-ts.svg`
Logomarca pura (sem wordmark). Dois hexagonos lado a lado:
- **Hex 1 vira T**: aresta superior = barra, haste central amarela = nucleo
- **Hex 2 vira S**: stroke tracando top→diagonal-esq→meio-horizontal→diagonal-esq→base.
  Meio horizontal amarelo = nucleo/transformacao

Outline dos hex fica sutil (opacity 35%) — quem protagoniza sao as letras
compostas. Amarelo marca o "coracao" de cada letra.

## Como escolher

- **01** se quer narrativa visual (entrada→nucleo→saida)
- **02** se quer simbolo forte que funciona como favicon/avatar isolado
- **03** se quer maximo minimalismo e espaco pro wordmark respirar

Cada um pode ser simplificado/intensificado. Diz qual dialoga melhor com
o que voce imagina e refinamos (ajuste de proporcao, peso, espacamento,
variantes mono/negativa etc).
