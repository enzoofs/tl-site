# Logo TimeLabs — V1 (3 direcoes)

Cada direcao explora um angulo do arquetipo **Alquimista Moderno**. Abra os SVGs no navegador pra visualizar.

## Direcao 1 — Hexagono + Ampulheta (sugerida)
`01-hexagono-ampulheta.svg`
Simbolo = hexagono com ampulheta estilizada dentro. Linhas finas, minimalista, conecta "lab" (hex) e "tempo" (ampulheta). Mais conceitual.

## Direcao 2 — Wordmark Puro Refinado
`02-wordmark-puro.svg`
Evolucao do logo atual. Mantem "time labs" lowercase, adiciona ponto amarelo ">" apos "labs" sugerindo fluxo/proximo-passo. Mais seguro e elegante.

## Direcao 3 — Monograma TL Hexagonal
`03-monograma-tl.svg`
Simbolo = "TL" geometrico dentro de hexagono com linha de fluxo saindo. Funciona como avatar/favicon. Mais corporate/escalavel.

## Como escolher
- Direcao 1: quer simbolismo forte e memoravel
- Direcao 2: quer continuidade com deck atual, minimo risco
- Direcao 3: precisa de icone/app/favicon que funciona sozinho

Diz qual direcao ressoa e a gente refina.
